## Polygon Amoy Testnet

### Public RPC Node

```bash
https://rpc-amoy.polygon.io/
```
### Network Details
- **Network Name**: Polygon Amoy Testnet
- **RPC URL**: `https://rpc-amoy.polygon.io/`
- **Chain ID**: 80001
- **Currency Symbol**: POL
- **Block Explorer URL**: [Polygon Amoy Explorer](https://explorer.amoy.polygon.io/)

### Faucet Options
To get test MATIC for the Polygon Amoy testnet, you can use any of the following faucets:
  * **[Option 1](https://faucet.polygon.technology/)**
  * **[Option 2](https://www.alchemy.com/faucets/polygon-amoy)**
  * **[Option 3](https://faucets.chain.link/polygon-amoy)**

### Adding to Wallet via Chainlist
1. Go to [Chainlist](https://chainlist.org/).
2. Search for **Polygon Amoy Testnet**.
3. Click on **Add to Wallet** and confirm the addition in your wallet.
